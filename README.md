### Description

Pipelines for data analysis and machine learning.

```
pipelitools/pipelitools
	|-- __init__.py  
	|-- preprocessing  
		|-- __init__.py  
		|-- eda.py  
		|-- feature_selection.py  
		|-- outliers.py  
	|-- models  
		|-- __init__.py  
		|-- classification.py  
		|-- metrics.py  
	|-- utils.py  
```

### Installation

```
pip install pipelitools
```
